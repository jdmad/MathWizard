MathChallenge

This is the first puzzle!

Your goal is to use the three blue buttons below to get values of:

  • 24 for the leftmost blue button
  • 15 for the middle blue button
  • 7 for the rightmost blue button

Clicking each of the three blue buttons has a specific effect:

  • the leftmost button adds to each button value (left to right: 2, 1, and 1)
  • the middle button subtracts from each button value (left to right: -5, -1, and -5)
  • the rightmost button doubles each button value (x2 for each)

Some other things to keep in mind:

  • You can reset all button values to zero with the green "Reset" button.
  • You can submit your solution with the green "Submit" button.
  • You can see a solution with the green "Show Solution" button.
  • You can show hints with the red "Hint 1", "Hint 2", and "Hint 3" buttons.

Good luck!
